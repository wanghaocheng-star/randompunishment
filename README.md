## 项目介绍
本工具是为课堂教学场景设计的**随机视频抽选程序**，核心逻辑是读取配置文件中的数字范围，随机生成一个数字，并自动打开 `video` 目录下对应编号的 MP4 视频文件。
适用于老师用「随机播放指定视频+表演」的趣味方式替代传统惩罚，操作极简、无需复杂配置，开箱即用。

### 核心功能
✅ **配置化数字范围**：通过 `config/randomscope` 文件自定义随机数的起始/结束范围，灵活适配不同数量的视频文件  
✅ **随机公平抽选**：基于 Python 原生 `random` 模块生成随机数，保证抽选公平性  
✅ **自动打开视频**：抽选完成后自动定位并打开 `video` 目录下对应编号的 MP4 文件  
✅ **轻量化无依赖**：仅依赖 Python 内置模块（`random`/`os`），无需安装任何第三方库  

### 适用场景
- 课堂随机表演：视频文件可存放表演示范、趣味任务等内容，抽中对应数字即完成视频相关的表演
- 课堂互动游戏：随机播放指定视频，让学生完成视频中的小任务
- 简易随机抽奖：适配任何需要「数字抽选+打开对应文件」的场景

## 快速开始
### 环境要求
- Python 3.6+（兼容所有主流Python版本，无需额外依赖）
### 部署与运行
- 下载项目
- 运行main.py
### 配置文件（必看）
- video文件夹放你需要播放的视频按数字顺序命名。后缀名为.mp4如1.mp4 2.mp4 3.mp4
- config文件夹下的randomscope文件第一行写想要播放视频的名字的最小值一般为1 第二行写最大值一般有多少个视频填多少
